# 伝統工芸品セレクトショップ「F.K. CODING & DESIGN」ポップバージョン LP制作

実在する店舗のリニューアルを想定し、AI技術を実務プロセスに組み込んで制作したトップページ制作プロジェクトです。
「奈良の伝統を、今の暮らしに。」をコンセプトに、伝統工芸の重厚さと、観光客が親しみやすいポップな世界観の両立を目指しました。

ブランドの伝統を堅実に守る方向性を追求したのが【モダン案】で、クライアントの『振り切ったポップさも見たい』という要望に応えたのが【本案】です。

## コンセプト

- **「伝統とモダンの融合（ポップ・アプローチ）」**
- 奈良の伝統工芸品（赤膚焼、一刀彫、仏像など）が持つ「静」のイメージを、AI生成による明るい配色で刷新しました。
- 企業実習の担当者からの「振り切ったポップな提案も見たい」という要望に応え、親しみやすさを最大化しています。

## 実装・こだわったポイント

### 1. AIを戦略的に活用したデザイン・プロセス

- **要件定義の自動化**: 旧サイトのコンテンツをGeminiに読み込ませ、現代のニーズに合わせた訴求ポイントを抽出。
- **ビジュアル生成**: 「奈良＝鹿」というアイコンを軸に、大仏と擬人化した鹿を組み合わせた独創的なHero画像をAIで生成し、唯一無二の世界観を構築しました。

### 2. ユーザビリティとパフォーマンスの最適化

- **レスポンシブ設計**: 「観光中のモバイル閲覧を想定し、JavaScriptによるハンバーガーメニューの実装と、親指で操作しやすいナビゲーションを設計。
- **表示速度の向上**: pictureタグによるWebP形式の採用、および画像へのloading="lazy"付与により、リッチなビジュアルと高速な読み込みを両立させました。

### 3. 商品の魅力を引き立てるグリッドレイアウト

- **視認性の高いカード設計**: 赤膚焼、一刀彫、奈良漆器などの各カテゴリーを直感的に判別できるよう、余白を活かしたグリッドレイアウトを採用し、情報の整理を行いました。

## 制作の背景と課題解決

職業訓練校の企業実習にて制作。店舗移転に伴う新サイト制作の「比較検討用プロトタイプ」という位置付けです。
「伝統を守りつつ、いかに新しい顧客層へアピールするか」という課題に対し、AIが提案する斬新なアイデアと、実務で求められる堅実なコーディングを掛け合わせることで、多角的な提案を実現しました。
